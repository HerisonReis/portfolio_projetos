Projetos em Análise de Dados

Hackday - Construção de um Dashboard Interativo com Power BI e SQL

Este projeto foi desenvolvido durante um Hackday intensivo (nos dias 25 e 26 de janeiro de 2025), onde apliquei conceitos avançados de ETL, modelagem de dados e lógica de negócios para transformar dados brutos em insights estratégicos. Utilizando SQL para a extração e transformação dos dados no PostgreSQL, estruturei um pipeline eficiente que alimentou um dashboard interativo no Power BI. O conjunto de dados públicos utilizado foi proveniente da BiblioTech.
O resultado final foi um painel dinâmico, composto por: Dashboard principal com análises gráficas intuitivas, visão detalhada em formato de tabelas dinâmicas e duas páginas adicionais com insights relevantes para stakeholders. .

As ferramentas utilizadas foram:

  • SQL
  • Power BI
  • PostgreSQL

Desenvolvimento de um Painel Gerencial para Marketplace de Entregas com o Python e Streamlit

Neste projeto, foram aplicados conceitos de programação em Python, manipulação de dados, pensamento estratégico e lógica de negócios, juntamente com ferramentas de desenvolvimento web, como Streamlit e GitHub. O objetivo foi desenvolver um painel gerencial que apresenta as principais métricas de desempenho de uma empresa de marketplace de comida ao redor de 15 Países.
O resultado final foi um painel hospedado em um ambiente na nuvem, acessível por meio de um link web. Compatível com qualquer dispositivo conectado à internet, o painel oferece praticidade e flexibilidade para análise de dados e suporte à tomada de decisões.
